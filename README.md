# library-starter

Features:

- Package manager [pnpm](https://pnpm.js.org/), safe and fast
- Execute TypeScript with [tsx](https://github.com/esbuild-kit/tsx)
- Bundle with [tsup](https://github.com/egoist/tsup)
- Test with [vitest](https://vitest.dev)
